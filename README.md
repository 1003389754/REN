# REN
Code for paper "Robust Ensembling Network for Unsupervised Domain Adaptation", by Han Sun, Lei Lin, Ningzhong Liu, and Huiyu Zhou.

## Prerequisites
- PyTorch >= 0.4.0 (with suitable CUDA and CuDNN version)
- torchvision >= 0.2.1
- Python3
- Numpy
- argparse
- PIL
